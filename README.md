# BOSS-RAID PVE 컨텐츠 API 서비스
09.16~09.22 `Nest.js` `BACKEND`

## 구현 개요
보스레이드 PVE 컨텐츠 관련 라우터 작성
* 유저 생성
  * 게임을 진행하는 사용자 생성
* 유저 조회
  * 사용자의 total score 조회
  * 사용자의 게임 history 조회
* 보스레이드 시작
  * 사용자가 보스레이드에 참가
  * 보스레이드 레벨 선택
  * 
* 보스레이드 상태 조회
  * 보스레이드
* 보스레이드 종료
* 랭킹 조회

## 사용기술
`Nest.js` `mySQL` `TypeORM` `TypeScript` `redis` `AWS` `elastiCache`
